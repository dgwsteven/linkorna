# LINKORNA MVP

Professional B2B SaaS MVP prototype for LINKORNA, a Cross-border AI Workforce Platform.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included screens

- Landing page
- Pricing page
- Dashboard
- Six AI employee task pages
- Mock task result page
- Login and register mock pages

The MVP currently uses structured mock outputs. It is prepared for Supabase Auth, task persistence, OpenAI task generation and Stripe subscriptions.

For deployment steps, see `DEPLOYMENT.md`.
